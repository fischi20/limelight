# Change Log

All notable changes to the "limelight" extension will be documented in this file

### 0.0.1

Initial release of the Limelight extension.
    - Block Focus
    - Selection Focus

### 0.0.2

- Fix: If the selection is on the first line of the file, the first line is not dimmed
- Fix: If the selection is only 1 line long, nothing is selected
- Feature: Set focus now drops the highlight if the range is the same as the previous range (applies to Ctrl+Alt+W)
